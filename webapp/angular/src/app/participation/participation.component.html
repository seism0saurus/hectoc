Participate in the project

There are multiple ways to improve the project. You can follow the Hourly Hectoc Bot on Mastodon. You can answer the challenges of the bot. You can tell other mental calculators about the project. You can inform me about issues on GitHub. You can ask for improvements on GitHub. You can write own improvements in a fork and send me a pull request on GitHub. You can send me mails with ideas or complaints.

Issues and Errors

See Bugs for a short description, how to tell me about errors or problems.

GitHub

The code this hectoc project is hosted on GitHub. Here is the GitHub repository.
You can open issues with ideas or send me pull requests (with description please) with your own implementations.
The project is licensed under the MIT License. So you are very free in the usage of the code.
Since the project is quite small at the moment, there are no tags or formalized procedures at the moment. Just reach out to me.

Mail

You can also send me mails to hectoc@seism0saurus.de, if you don't have a GitHub account.
